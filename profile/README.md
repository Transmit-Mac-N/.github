## Overview

Transmit Advanced is the premier file transfer client from Panic that connects to remote servers and cloud storage services via SFTP, FTP, Amazon S3, Google Cloud Storage, Azure, Backblaze B2, Rackspace, and other protocols with the polish and performance that Panic's two decades of macOS development expertise produce. It provides a native dual-pane file browser where local and remote files are managed side by side with the same drag-and-drop, Quick Look, and keyboard shortcut conventions that Finder uses, making remote file management feel as natural as local file management.

Protocol support covers the full range of file transfer and cloud storage protocols including SFTP, FTP, FTP with TLS, Amazon S3, Google Cloud Storage, Microsoft Azure, Backblaze B2, Rackspace Cloud Files, DreamObjects, and WebDAV. Each protocol is implemented with full feature support including server-specific capabilities like S3 storage classes, lifecycle policies, and access permissions. The connection manager stores server credentials securely in the system keychain.

The dual-pane interface displays local and remote file systems side by side with synchronized navigation that makes comparing local and remote directory structures intuitive. File operations — copy, move, rename, delete, create directory — work identically on local and remote files. Quick Look previews remote files without downloading them, and Get Info displays detailed file metadata including permissions, modification dates, and server-specific properties.

Sync and upload tools compare local and remote directories and synchronize their contents, enabling deployment workflows that upload changed files while leaving identical files untouched. The sync engine offers preview before execution, showing exactly which files will be uploaded, downloaded, or deleted. Rules exclude specific file types, names, or patterns from sync operations.

For web developers, system administrators, and anyone who manages files on remote servers or cloud storage, Transmit Advanced is the file transfer client that makes remote file management as intuitive and polished as local file management. Panic's commitment to native macOS design produces a tool that feels like a natural extension of the operating system.

## Performance

Transmit balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
